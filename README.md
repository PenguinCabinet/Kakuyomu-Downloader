# Kakuyomu Downloader
これはカクヨム非公式ダウンローダーです。   
**カクヨム利用規約違反の可能性があるため、運用しておりません。**    
**運用する場合は運用に生じたすべての責任は運用者の自己責任です。製作者は一切の責任を負いません**

# Dev
Severless Frameworkを使用しています。   
インフラはAWS+Github Pageを使用しています。

## Backend
### Debug
ローカルで実行。
```shell
sls offline
```
### Deploy
```shell
sls deploy
```

## Frontend
### Debug
```shell
cd docs
python -m http.server
```
